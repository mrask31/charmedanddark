# Shop Images

## Required Images

### Hero Image
- **Filename**: `hero.jpg`
- **Dimensions**: 1920x1080 (or similar 16:9 ratio)
- **Description**: Atmospheric gothic/dark aesthetic image for The Atelier hero section
- **Usage**: `components/shop/ShopHero.js`

## Placeholder
Until the hero image is added, the component will show a dark background with the hero text overlay.

## Adding Images
1. Place `hero.jpg` in this directory (`public/images/shop/`)
2. The image will be automatically loaded by the ShopHero component
